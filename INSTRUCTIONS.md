## Key Differences from Python

| Python | JavaScript | 
|--------|-----------|
| `def greet(name):` | `function greet(name) {` |
| `self.name = name` | `this.name = name` |
| `True / False / None` | `true / false / null / undefined` |
| `print("hello")` | `console.log("hello")` |
| `# comment` | `// comment` |
| `len(list)` | `array.length` |
| `list.append(x)` | `array.push(x)` |
| Indentation matters | Curly braces `{}` matter |
